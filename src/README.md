#无人系统组大作业仓库  
这是无人系统组实习第二组大作业的仓库，在使用该仓库的内容前请确保以下系统和软件版本正确：  
1.Ubuntu 22.04 LTS  
2.ROS2 Humble  
3.Gazebo Harmonic    
  
#使用指南：  
1.在主目录下创建一个名为scutracing_ws的工作空间，将本仓库克隆到src文件夹(当然也可以是其他路径，为避免路径问题最好是这样创建)  
2.因为使用了绝对路径，所以要改掉所有.launch.py .sdf等文件中的路径（只记得这几个，可能还有其他的，运行时报错根据报错信息再改） 
3.退回scutracing_ws,终端中执行colcon build --symlink-install编译（第一次会比较久）  
4.source环境，执行常用命令.txt中的命令  


#注意事项  
1.一定一定一定要确保使用的是Gazebo Harmonic！！！ROS2 Humble默认的不是这个！！！所以需要ros_gz桥接  
2.所有python代码均使用4空格进行缩进，不要使用Tab  
3.命名只使用字母、数字和下划线，否则有可能报错，同时命名要做到见名知义
